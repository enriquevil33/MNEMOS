Core Capabilities

A complete knowledge management system with advanced AI-powered features for deep understanding and semantic navigation

Multimodal Processing

Ingest and analyze content from any source with intelligent extraction and transcription.

GraphRAG & Reasoning Engine

Navigate complex knowledge structures and discover non-obvious connections between concepts.

Hybrid Search System

Combine the best of vector similarity and keyword matching for precise retrieval.

Flexible AI Integration

Connect to any LLM provider or run models locally with complete control.

Technology Stack

Built on proven, modern technologies for performance, scalability, and reliability

Frontend

  • Angular 19 (SPA)
  • TailwindCSS
  • RxJS
  • Cytoscape.js (Graph viz)

Backend

  • Flask (Python)
  • SQLAlchemy ORM
  • Celery (Async tasks)
  • Redis (Queue)

Database

  • PostgreSQL 16
  • pgvector (Vector search)
  • HNSW indices
  • GIN indices (FTS)

AI & ML

  • OpenAI Whisper
  • sentence-transformers
  • LangChain
  • Multiple LLM APIs

Use Cases

Research & Academia

Analyze research papers, extract key concepts, and discover connections across your entire academic library with intelligent semantic search.

Enterprise Knowledge Base

Build a centralized knowledge repository from documents, meetings, and training materials with persistent memory and context.

Learning & Education

Process lecture videos, PDFs, and notes to create an intelligent study assistant that understands relationships between topics.

Technical Documentation

Navigate complex technical documentation with hypergraph reasoning to understand dependencies and architectural relationships.

Quick Start

# Clone the repository
git clone https://github.com/qepri/MNEMOS.git
cd MNEMOS

# Install Docker Desktop (required)
# Windows/Mac: https://www.docker.com/products/docker-desktop

# Run the one-click installer (Windows)
installer.bat

# Or use Docker Compose (Linux/Mac/Windows)
docker-compose up -d

# Access the web interface
http://localhost:5200
